# Prompt para Logo SafariLink

## Prompt Principal

```
Diseña un logo moderno y minimalista para "SafariLink", una plataforma Web3 de hackathons. El logo debe combinar elementos de safari africano con tecnología blockchain. Incluye un león estilizado o silueta de león integrada con elementos tecnológicos (circuitos, bloques de blockchain, o conexiones de red). Colores: naranja/dorado safari (#FF6B35, #F7931E) combinado con azul tecnológico (#0066FF, #00D4FF). Estilo: glassmorphism, moderno, profesional. El diseño debe funcionar en modo claro y oscuro, y ser reconocible en tamaños pequeños. Sin texto, solo símbolo/icono.
```

## Variaciones del Prompt

### Versión Minimalista
```
Logo minimalista para SafariLink: silueta de león estilizada que forma una "S" o se integra con símbolos de conexión/red. Colores: naranja (#FF6B35) y azul (#0066FF). Estilo moderno, tech-forward, funciona en fondos claros y oscuros.
```

### Versión Tech-Focused
```
Logo para SafariLink: león digitalizado con elementos de blockchain (bloques conectados, nodos de red). Colores: gradiente naranja-dorado a azul cian. Estilo: glassmorphism, futurista, Web3. Icono único sin texto.
```

### Versión Simple
```
Logo SafariLink: león + conexión/red. Colores naranja y azul. Minimalista, moderno, tech.
```
